入圍放視作品-Part3

2023/5/1

教學關卡-1 : 場地上有多隻小怪，系統會提示玩家如何操作

教學關卡-2 : 玩家進場後會看到影子怪擋住去路，玩家必須使用攻擊擊敗牠

教學關卡-3 : 玩家利用特殊怪物的衝擊波暈眩敵人，玩家必須要運用攻擊跟迴避通關

關卡 BOSS-1 : 光神的騎士擁有堅硬的護盾，玩家需將光神的騎士引誘到特殊怪物的旁邊， 利用爆炸擊碎它的護盾

關卡 BOSS-2 : 場地四周會有裝置並隨機在任意裝置上產生特殊的能量體，玩家必須在被干擾的情況下，
前去收集，並放進中間的發射口，讓影子消失，來達成玩家可以攻擊王的條件

關卡-3 最終 BOSS : 玩家需要一次面對兩隻 Boss，兩隻 Boss 的攻擊會互相傷害到對方，玩家必須巧妙地調整站位，
這樣就能使兩方的攻擊互相打到對方，以此來擊敗兩位 Boss
